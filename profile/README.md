# Janustack

### Monorepos
- [Janush](https://github.com/janustack/janush)
- [XML](https://github.com/janustack/xml)
- [SVG](https://github.com/janustack/sax)
- [Signadex](https://github.com/janustack/signadex)

### Libraries

- [config](https://github.com/janustack/config) - Configuration loader for Bun and browser environments
- [css-selector-parser](https://github.com/janustack/css/packages/selector-parser) - CSS selector parser written in Zig with WASM support
- [xml](https://github.com/janustack/xml) - Monorepo for XML tooling and utilities
    - [@janustack/sax](https://github.com/janustack/xml/packages/sax) - High-performance SAX parser for XML, HTML, and JSX
    - [@janustack/xml-utils](https://github.com/janustack/xml/packages/xml-utils) - XML 1.1 utility functions 
- [svg](https://github.com/janustack/svg) - Monorepo for SVG tooling and utilities
    - [@janustack/svg-cli](https://github.com/janustack/svg/packages/cli) - Command-line interface for SVG optimization and transformation
    - [@janustack/svg-components](https://github.com/janustack/svg/packages/components) - Convert SVG files into framework components
    - [@janustack/svg-optimizer](https://github.com/janustack/svg/packages/optimizer) - Plugin-based SVG optimization engine

### Templates

- [ts-starter-monorepo](https://github.com/janustack/ts-starter-monorepo) - Monorepo template that utilizes Bun and Moonrepo
